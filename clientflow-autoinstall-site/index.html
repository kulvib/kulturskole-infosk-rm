ClientFlow install v5.4

Ny installation bruger kun enrollment/client-secret-flow.

Ubuntu on Xorg, automatisk login, fuld systemopdatering og kiosk-hardening sættes automatisk. Genstart kræves efter installation.

Efter login på den nye Ubuntu-klient:

sudo apt-get update
sudo apt-get install -y curl unzip ca-certificates
curl -fL https://autoinstall.onrender.com/setup-clientflow-desktop-icon.sh -o setup-clientflow-desktop-icon.sh
chmod +x setup-clientflow-desktop-icon.sh
bash setup-clientflow-desktop-icon.sh

Derefter åbnes skrivebordsikonet ClientFlow Installer. Installeren viser trin undervejs, så længere opdateringer ikke ligner en frysning.

GUI: Netværksinfo viser aktiv forbindelse, og kalenderen viser 5 dage frem.

1) Installer ClientFlow med installationskode
2) Indtast ny installationskode / re-registrer klient
3) Geninstaller/opdater ClientFlow
4) Verificér installation
5) Vis logs/status
