# Kulturskole Infoskærm

Dette repository indeholder hele projektet til styring og overvågning af Kulturskolens infoskærme:

- **backend/**: Python (Flask/FastAPI) backend til Render.com (REST API, WebSocket, JWT)
- **frontend/**: React (eller Vue) dashboard til Vercel.com
- **client-agent/**: Python-systemd agent til Ubuntu-klienter

## Quickstart

1. Se `backend/README.md` for backend-setup (Render)
2. Se `frontend/README.md` for frontend-setup (Vercel)
3. Se `client-agent/README.md` for opsætning på Ubuntu-klienter
